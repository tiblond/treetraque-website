# Treetraque Website

Bienvenue sur le projet Treetraque, un site web statique conçu pour fournir des informations sur nos services et notre mission. Ce projet utilise HTML5, CSS3 et JavaScript Vanilla.

## Structure du projet

Voici la structure des dossiers et fichiers du projet :

```
treetraque-website
├── index.html
├── css
│   ├── style.css
│   ├── normalize.css
│   └── responsive.css
├── js
│   ├── main.js
│   ├── navigation.js
│   └── utils.js
├── images
│   └── .gitkeep
├── pages
│   ├── about.html
│   ├── participate.html
│   └── contact.html
├── assets
│   ├── fonts
│   │   └── .gitkeep
│   └── icons
│       └── .gitkeep
├── .gitignore
└── README.md
```

## Technologies utilisées

- **HTML5** : Pour la structure du contenu.
- **CSS3** : Pour le style et la mise en page.
- **JavaScript** : Pour l'interactivité.

## Installation

1. Clonez le dépôt : `git clone <url-du-depot>`
2. Ouvrez le fichier `index.html` dans votre navigateur pour voir le site en action.
